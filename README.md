### How To Use ?
1. First Download Script<br/>
2. Install All Requirement Modules(I Wrote `requirements.txt`)<br/>
3. Finally Use It

#### Before Run
```
# pip3 install -r requirements.txt
```

#### Run
```
# python3 fire_Ball.py
```
